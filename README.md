# Hackathon Todo App - Frontend & CLI

This repository contains the frontend and CLI components of the Hackathon Todo App.

## Projects

### 1. Frontend (Next.js)
The frontend is a modern task management dashboard built with Next.js, Tailwind CSS, and Recharts.

**Deployment to Vercel (EASY):**
1. Connect this repository to Vercel.
2. Vercel will automatically detect the project settings because the code is in the root.
3. **Important**: Set the Environment Variable `NEXT_PUBLIC_API_BASE_URL` to your deployed backend URL.
4. Click **Deploy**.

### 2. CLI Project (Python)
A console-based interface for managing tasks.

**Location:** `cli/` folder.
**Setup:**
- Navigate to the `cli` folder.
- Run `python src/todo_app.py`.

## Note
This repository does NOT include the backend code. Ensure your backend is deployed and accessible via the API URL configured in the frontend.
